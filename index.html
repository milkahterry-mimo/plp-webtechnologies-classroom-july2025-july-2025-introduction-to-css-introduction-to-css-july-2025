About This Page

This page demonstrates how margins, padding, and borders work together in the CSS Box Model.

Box Model Basics

Every element in CSS is a oval box made up of content, padding, border, and margin.

Padding & Margin

Padding adds space inside the border; margin adds space outside it.

Borders

Borders surround the padding and content, defining the edge of an element’s box.
